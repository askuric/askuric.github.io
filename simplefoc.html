SimpleFOC project

Arduino Compatible
Open Source
Field Oriented Control library

The aim of the SimpleFOC library to demistify the Field Oriented Control (FOC) algorithm and make a robust but simple implementation for Arduino hadrware, as well as to introduce the new Arduino FOC shield board.

YouTube demonstration video

This is a short video shownig the setup procedure and the capabilites of the SimpleFOC algorithm.

The video setup consists of:

  • HMBGC V2.2 board Ebay link
  • AMT 103 CUI Encoder 2048ppr Mouser link
  • BLDC Gimbal Motor Aliexpress link

Features

Arduino Compatible

SimpleFOC algorithm is written and optimised for usage with Arduino boards.
Tested boards till this moment are:

  • Arduino UNO, Arduino MEGA,
  • HMBGC V2.2
  • STM32 Bluepill (minor modifications)
  • ESP32 (minor modifications)
The SimpleFOC library is available through the Arduino Library Manager.

Easy configuration

SimpleFOC is very simple to configure for different types of BLDC motors, position sensors and microcontrollers.

Open Source

The code and full documentation is available on Github.

Modular

The aim of this project is to provide a good base for the community to run the FOC algorithm for as many hardware configurations and application as possible.

Arduino SimpleFOC Shield board

Low cost modular BLDC motor driver board

  • Plug & play

    Fully compatible with SimpleFOC library and most of Arduino boards

  • Supports stacking

    Possible stacking 2 borads (2 BLDC motors + 2 sensors)

  • Configurable pinout

    Pin configuration in hardware for L6234 and Encoders

  • Encoder Pullups

    Integrated configurable 3KOhm Encoder pullups

  • Low cost

    $25-$40
