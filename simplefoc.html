Features

Arduino Compatible

SimpleFOC library is written and optimized for usage with Arduino boards.
Tested boards until this moment are:

  • Arduino UNO, Arduino MEGA,
  • HMBGC V2.2
  • STM32 Bluepill (minor modifications)
  • ESP32 (minor modifications)
The SimpleFOC library is available through the Arduino Library Manager.

Easy configuration

SimpleFOC is very simple to configure for different types of BLDC motors, position sensors and microcontrollers.

Open Source

The code and full documentation is available on Github.

Modular

The aim of this project is to provide a good base for the community to run the FOC algorithm for as many hardware configurations and application as possible.

Arduino SimpleFOCShield

Low cost modular BLDC motor driver board

  • Plug & play

    Fully compatible with SimpleFOC library and most of the Arduino boards

  • Stackable

    Possibility of stacking 2 shields (2 BLDC motors + 2 position sensors)

  • Configurable pinout

    Possible to choose all the pins used by the board.

  • Encoder Pull-Ups

    Integrated configurable 3KOhm Encoder pull-ups

  • Open-Source

    Schematics and fabrication files fully available.
