Arduino SimpleFOClibrary

Arduino Field Oriented Control Library

The aim of this project is to demystify the Field Oriented Control algorithm and create a good base for the wide use of BLDC motors in the low-cost community.

  • Arduino Compatible

    Fully compatible with most of the Arduino boards

  • Fully configurable

    Easy to setup and configure for different hardware and applications.

  • Modular

    Applicable to any type of BLDC motor + driver combination, supports Encoders and Magnetic sensors

Arduino SimpleFOCShield

Low cost modular BLDC motor driver board

  • Plug & play

    Fully compatible with SimpleFOC library and most of Arduino boards

  • Supports stacking

    Possible stacking 2 boards (2 BLDC motors + 2 sensors)

  • Configurable pinout

    Pin configuration in hardware for L6234 and Encoders

  • Encoder Pullups

    Integrated configurable 3KOhm Encoder pullups

  • Low cost

    $25-$40

Inertia Wheel Inverted Pendulum project

Open source 3d printed inverted pendulum setup

This project is suitable for anyone with extra free time and passion for interesting projects, but especially for electrical engineering students (control theory students) to learn the control approach of an unstable process and test their algorithms, the primary contorller used in this project is LQR.

  • STM32 code

    Written for STM32 Nucleo board (Arduino headers)

  • CAD designs

    All CAD designs available for modification and STL files provided

  • Multiple control strategies

    PID and LQR loops implemented

  • Plug and Play

    Easy setup
