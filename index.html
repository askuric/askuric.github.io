Arduino SimpleFOClibrary

The aim of this project is to demystify the Field Oriented Control algorithm and create a good base for the wide use of BLDC motors in the low-cost community. This Arduino library implements Field Oriented Control (FOC) algorithm for BLDC motors. FOC algorithm produces incomparably smooth operation and high degree of torque, velocity and position control. The library is intended both for:

  • Beginners searching for a simple and user-friendly way to learn how to control BLDC motors
  • Advanced users ready to dig deeper into the FOC algorithm and optimize the code for their particular application/hardware.

  • Arduino Compatible

    Fully compatible with most of the Arduino boards
    Available through Arduino Library Manager

  • Fully configurable

    Easy to setup and configure for different hardware and applications.

  • Modular

    Applicable to any type of BLDC motor + driver combination, supports Encoders and Magnetic sensors
    Supports Arduino, STM32 and ESP32 boards

  • Plug & play

Arduino SimpleFOCShield

This is an open-source low-cost Brushless DC (BLDC) motor driver board intended primarily for low-power FOC applications up to 5Amps. The board is fully compatible with the Arduino UNO and all the boards with the standard Arduino headers. The SimpleFOCShield, in combination with the SimpleFOClibrary provides user-friendly way to control BLDC motors both in hardware and software.

  • Plug & play

    Fully compatible with SimpleFOClibrary and any board with Arduino UNO headers.

  • Stackable

    Possibility of stacking 2 shields (2 BLDC motors + 2 position sensors)

  • Configurable pinout

    Possible to choose all the pins used by the board.

  • Encoder Pull-Ups

    Integrated configurable 3KOhm Encoder pull-ups

  • Open-Source

    Schematics and fabrication files fully available.

Arduino FOC Inertia Wheel Inverted Pendulum

This is a project of designing and controlling the reaction wheel inverted pendulum based entirely on Arduino SimpleFOC library and SimpleFOC shield.
This is a very fun project in many ways, and it is intended:

  • Students in search for a good testing platform for their advanced algorithms
  • Everyone with a bit of free time and a motivation to create something cool :D

  • Arudino code

    Written for Arduino UNO using SimpleFOClibrary and SimpleFOCShield

  • CAD designs

    All CAD designs available for modification and STL files provided

  • Control algorithm

    State-space LQR controller

  • Plug and Play

    Very simple to assemble, configure and setup.
